Temperature Controlled Fan Regulator

A closed-loop thermal control system using an Arduino UNO and LM35 sensor that automatically adjusts fan speed relative to temperature. Written in C++ to create a linear relationship, dynamically controlling fan duty cycles across a 20°C–60°C range using PWM signals and NPN transistors.

OPA Alice Condenser Microphone

A custom XLR condenser microphone built from scratch, converting 48V phantom power into stable 12VDC using zener diodes, low-pass filters, and decoupling capacitors. A non-inverting op-amp amplifier removes static for clean audio output. PCB designed in KiCAD and hand-soldered with debugging validation.

Electronics Signal Filtering PCB Design KiCAD Soldering

Keyboard Prototype

A fully functional mechanical keyboard built using an STM32 microcontroller, matrix column-row switch layout, and a C++ keyboard template library for switch binding. After troubleshooting, achieved sub-2ms input latency — on par with standard gaming keyboards.
